p (X ≥ 3)

p (X ≥ 3) = 1 - p(X < 3) = 1 - 0.4 = 0.6

p (3 ≤ X < 4.5)

p (3 ≤ X < 4.5) = p (X < 4.5) - p(X < 3) = 0.9 - 0.4 = 0.5
