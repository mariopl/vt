Ejercicio 7 resuelto

2x + y ≤ 3

2x + y = 3

x = 0;     2 · 0 + y = 3;   y = 3;          (0, 3)

x = 1;     2 · 1 + y = 3;   y = 1;          (1, 1)

2 · 0 + 0 ≤ 3       0 ≤ 3      Sí

Ejercicio 8 resuelto

2x + y > 3

2 · 0 + 0 > 3       0 > 3      No

En este caso los puntos de la recta no pertenecen a la solución.
