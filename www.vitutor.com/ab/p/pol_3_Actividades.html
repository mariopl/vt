Ejercicio 5 resuelto

Dados los polinomios:

P(x) = 4x2 − 1

Q(x) = x3 − 3x2 + 6x − 2

R(x) = 6x2 + x + 1

S(x) = 1/2x2 + 4

T(x) = 3/2x2 + 5

U(x) = x2 + 2

Calcular:

1P(x) + Q (x) =

= (4x2 − 1) + (x3 − 3x2 + 6x − 2) =

= x3 − 3x2 + 4x2 + 6x − 2 − 1 =

= x3 + x2 + 6x − 3

2P(x) − U (x) =

= (4x2 − 1) − (x2 + 2) =

= 4x2 − 1 − x2 − 2 =

= 3x2 − 3

3P(x) + R (x) =

= (4x2 − 1) + (6x2 + x + 1) =

= 4x2 + 6x2 + x − 1 + 1 =

= 10x2 + x

42P(x) − R (x) =

= 2 · (4x2 − 1) − (6x2 + x + 1) =

= 8x2 − 2 − 6x2 − x − 1 =

= 2x2 − x − 3

5S(x) + T(x) + U(x) =

= (1/2 x2 + 4 ) + (3/2 x2 + 5 ) + (x2 + 2) =

= 1/2 x2 + 3/2 x2 + x2 + 4 + 5 + 2 =

= 3x2 + 11

6S(x) − T(x) + U(x) =

= (1/2 x2 + 4) − (3/2 x2 + 5) + (x2 + 2) =

= 1/2 x2 + 4 − 3/2 x2 − 5 + x2 + 2 =

= 1
