Definición

Dadas dos fracciones algebraicas, reducirlas a común denominador es encontrar dos fracciones algebraicas equivalentes con el mismo denominador.

Pasos para reducir a común denominador

Nos valdremos de las fracciones siguientes:

1.Descomponemos los denominadores en factores para hallarles el mínimo común múltiplo, que será el común denominador.

x2 − 1 = (x + 1) · (x − 1)

x2 + 3x + 2 = (x +1 ) · (x + 2)

m.c.m. (x2 − 1, x2 + 3x + 2) = (x + 1) · (x − 1) · (x + 2)


2.Dividimos el común denominador entre los denominadores de las fracciones dadas y el resultado lo multiplicamos por el numerador correspondiente.
