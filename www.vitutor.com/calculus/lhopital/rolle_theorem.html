Rolle’s Theorem

If a function is:

Continuous on [a, b].

Differentiable on (a, b).

f(a) = f(b).

Then, there exists a point, c (a, b) such that f'(c) = 0.

The graphical interpretation of Rolle's Theorem states that there is a point where the tangent is parallel to the x-axis.
