Properties of Limits

1The limit, if it exists, is unique.

2If a sequence, an, has a limit, all the subsequences have the same limit as an.

3 All convergent sequences are bounded.

4There are bounded sequences that are not convergent.

5 All the monotone and bounded sequences are convergent.

6There are convergent sequences that are not monotonous.


Rules of Limits

lim (an + bn) = lim (an) + lim (bn)

lim (an − bn) = lim (an) − lim (bn)

lim (an · bn) = lim (an) · lim (bn)

lim (an : bn) = lim (an) : lim (bn)

lim k · an =k · lim an

lim ank = (lim an)k

lim loga an = loga lim an
