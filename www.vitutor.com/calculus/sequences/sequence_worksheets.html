Calculate the nth term of the following sequences:

1  8, 3, −2, −7, −12, ...

3 − 8 = −5.

−2 − 3 = −5.

−7 − (−2) = −5.

−12 − (−7) = −5.

d = −5.

an = 8 + (n − 1) (−5) = 8 −5n +5 = −5n + 13


2  3, 6, 12, 24, 48, ...

6/3 = 2.

12/6 = 2.

24/12 = 2.

48/24 = 2.

r = 2.

an = 3 · 2 n−1
