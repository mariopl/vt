The quadratic equation y = ax² + bx + c passes through the points (1,1), (0, 0) and (−1,1). Calculate the value of a, b and c.

0 = a · 0² + b · 0 + c

1 = a · (−1)² + b · (−1) + c

a = 1 b = 0 c = 0
