Solved Polynomial Word Problems

1

Find a and b if the polynomial x5 − ax + b is divisible by x2 − 4.

x2 − 4 = (x +2) · (x − 2)

P(−2) = (−2)5 − a · (−2) + b = 0

−32 +2a +b = 0         2a +b = 32

  P(2) = 25 − a · 2 + b = 0

32 − 2a +b = 0          − 2a +b = −32
