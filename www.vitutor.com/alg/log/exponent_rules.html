Product Rule

am · a n = am+n

25 · 22 = 25+2 = 27

Quotient Rule

am : a n = am - n

25 : 22 = 25 - 2 = 23

Power Rule

(am)n=am · n

(25)3 = 215 

Same Exponent

an · b n = (a · b) n

23 · 43 = 83

an : b n = (a : b) n

63 : 33 = 23
