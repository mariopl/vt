Ejercicio 1 resuelto

Calcular los coeficientes de la función f(x)= ax + b si f(0) = 3 y f(1) = 4.

1 Representar la función.

f(0) = 3

3 = a · 0 + b              b = 3

f(1) = 4.

4 = a · 1 + b               a = 1

f(x) = x + 3

2 Indicar los intervalos en los que es positiva o negativa.

x + 3 = 0 x = − 3

f(−4) = −1 < 0f(0) = 3 > 0

f(x) < 0 si x < −3

f(x) > 0 si x > −3
