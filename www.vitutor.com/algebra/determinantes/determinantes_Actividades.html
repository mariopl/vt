Ejercicio 13 resuelto

Resolver las siguientes ecuaciones matriciales:

1A · X = B

|A|=1 ≠ 0, existe la matriz inversa A−1 .

A−1 (A · X) = A−1 · B

( A−1 · A) · X = A−1 · B

I · X = A−1 · B

X = A−1 · B

 2  X · A + B = C

|A| = 1 ≠ 0

(X · A + B) − B = C − B

X · A + (B − B) = C − B

X · A + 0 = C − B

X · A = C − B

X · A · A−1 = ( C − B) · A−1

X (A · A−1 ) = ( C − B) · A−1

X · I = ( C − B) · A−1

X = ( C − B) · A−1
