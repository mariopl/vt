Ejercicio 1 resuelto

1Dadas las matrices:

Resolver la ecuación:

A · X = B

|A|=1 ≠ 0, existe la matriz inversa A−1 .

A−1 (A · X) = A−1 · B

(A−1 · A) · X = A−1 · B

I · X = A−1 · B

X = A−1 · B

Ejercicio 2 resuelto

Dadas las matrices:

Resolver la ecuación:

X · A + B = C

|A| = 1 ≠ 0

(X · A + B) − B = C − B

X · A + (B − B) = C − B

X · A + 0 = C − B

X · A = C − B

X · A · A−1 = (C − B) · A−1

X (A · A−1) = (C − B) · A−1

X · I = (C − B) · A−1

X = (C − B) · A−1

Ejercicio 5 resuelto

Resolver las ecuación matricial:

A · X + 2 · B = 3 · C

|A| = 1 ≠ 0

(A · X +2 · B) − 2 · B = 3 · C − 2B

A · X + (2 · B − 2 · B) = 3 · C − 2B

A· X + 0= 3 · C − 2B

A· X = 3 · C − 2B

(A−1 · A) · X = A−1 · (3 · C − 2B)

I · X = A−1 · (3 · C − 2B)

X = A−1 · (3 · C − 2B)
